# 记忆星图生成艺术宣言 · Memory Constellation Manifesto

## 一、遗忘不是消失，是稀释

1885 年，赫尔曼·艾宾浩斯在波恩的一间阁楼里背诵无意义音节，绘制出人类认知史上第一条遗忘曲线：`R = e^(-t/S)`。记忆并非在某一刻断裂，而是像墨滴落入清水——浓度递减，却始终存在。我们的艺术追随这条曲线。画面中的每一颗星点都是一段记忆痕迹（memory trace），它的亮度随时间衰减，它的连线随遗忘松散。但我们不视衰减为失去，而视之为稀释——记忆从浓烈的实体退化为弥散的底色，为新的痕迹腾出空间。

## 二、间隔重复即是呼吸

SM-2 算法告诉我们：在记忆即将稀释至临界点之前的一次复习，能让曲线重置并变得更陡。这不是简单的"记忆"，而是"呼吸"。我们的粒子系统以此为节律——每一次间隔重复，如同一次吸气，让即将熄灭的星点重新聚拢、增亮、重新编织进网络。复习越及时，吸气越深；间隔越长，星点在重新点亮前漂得越远。艺术因此获得了心跳。

## 三、知识即是星座

孤立的知识点会遗忘，但编织进网络的知识会长存。这是双链笔记与知识图谱的根本信念。在我们的画布上，当两个粒子足够接近，它们之间会诞生一条连线——细若蛛丝，却足以抵抗遗忘的稀释。连线越多，网络越密，星点越亮。一个高度连通的节点会自然成为"枢纽星"，发出琥珀色的微光，正如学习者脑中那些举一反三的核心概念。星座不是被画上去的，是从随机中生长出来的。

## 四、熵是美的刻度

我们拒绝规则的对称。Perlin 噪声场驱动粒子的漂移方向，使得每一帧的轨迹都不可重复，却始终连贯。这是记忆的真实形态——它从不整齐排列，它总是模糊的、偏移的、带有个人色彩的。我们刻意保留噪声的纹理，刻意让某些星点偏离预期轨道。因为完美是遗忘的前兆，而微小的混乱才是记忆存活的方式。

## 五、留白即休息

晨雾纸页的美学告诉我们：不是所有空间都需要填满。画面中的大面积留白不是空虚，而是"未学习"的状态——是等待新知识降临的沃土。我们用暖白色的底色承载星图，让记忆的痕迹在纸上呼吸，而非在黑暗中灼烧。琥珀色作为唯一的暖调点缀，只出现在最坚固的记忆和最活跃的连线上——它是被"掌握"的勋章。

**概念种子**: 艾宾浩斯遗忘曲线 `R = e^(-t/S)`——将记忆保持率映射为粒子亮度，将间隔重复映射为粒子复苏脉冲，将知识网络映射为动态星座连线。

**技术实现**: p5.js · Perlin 噪声流场 · 距离阈值连线 · 指数衰减亮度模型 · 种子化确定性随机
